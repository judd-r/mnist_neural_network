# Project description
Project deployed at https://mnist-neural-network.herokuapp.com/


This project visualises all layers of a Tensorflow Neural Network model predicting handwritten digits from the MNIST dataset.


Stack:
Python, Jupyter
Flask,
Streamlit,
Git,
Heroku
